Silver DJ media structure

Expected folders:
- gallery/
- logo/
- videos/

Suggested usage:
- gallery/: press photos and stills
- logo/: transparent logo assets
- videos/: local mp4 exports and posters
- youtube/: notes, thumbnails, or source exports if needed
